## AI モデルガバナンス

1. OpenAI Compatible、Responses、Claude Messages、Gemini、Realtime など複数のプロトコルに対応した統一モデル入口を提供します。
2. OpenAI、Azure OpenAI、Claude、Gemini、AWS Bedrock、Vertex AI、Ollama、DeepSeek、通義千問、智譜 GLM、Kimi、Doubao、Tencent Hunyuan、Wenxin、iFLYTEK Spark、MiniMax、01.AI、SiliconFlow などのチャネルを含む、複数プロバイダーのモデルプールに対応します。
3. モデル一覧、モデルマッピング、ユーザーグループ、トークングループ、モデル制限、価格ルールの管理をサポートします。
4. チャネル編集画面で、`chat/completions`、`responses`、`embeddings`、`rerank`、`video tasks`、モデル発見などの能力状態を能力マトリクスとして表示します。
5. API Key、モデル一覧、Base URL、JSON 設定、Vertex AI リージョン、Codex 認証情報、動画タスクのプレースホルダーなど、よくある設定リスクを検証できます。

## AgentOps と組織ガバナンス

1. Agent、ワークフロー、プラグイン、ツール呼び出し、ユーザーごとに独立した API Key を作成できます。
2. トークンのモデル範囲、クォータ制限、有効期限、グループ制御をサポートします。
3. リクエストログにはモデル、チャネル、グループ、ユーザー、トークン、消費量、レイテンシ、エラー、リトライ情報が記録され、Agent 呼び出し失敗やコスト異常の調査に利用できます。
4. 管理者はコンプライアンス要件に応じてログ監査を有効化できます。一般ユーザーのログ API では管理者専用の監査フィールドがフィルタリングされます。
5. ユーザー管理、グループ管理、引換コード、決済、サブスクリプション、ダッシュボード、システム設定など、組織運用の入口を提供します。

## 課金とコスト計算

1. 従来のモデル倍率、グループ倍率、チャネル倍率、リクエスト単位課金、キャッシュヒット課金をサポートします。
2. 式ベース課金により、段階価格、キャッシュ、画像 / 音声 token、時間帯割引、パラメータ加算を一つの式で管理できます。
3. `Tiered billing JSON` により、複数モデルの段階課金ルールをまとめて管理できます。
4. 動画などの非同期タスク向けに、プロバイダー、モデル、長さ、解像度、音声有無、動画入力有無などのフィールドで単価を照合するパラメータ化 rate-card をサポートします。
5. 長時間の Agent 呼び出しチェーンや非同期マルチモーダルタスクに適した、事前課金、失敗時返金、消費ログ、使用量統計をサポートします。

## マルチモーダルとタスクプロトコル

1. チャット、補完、画像、動画、音声、埋め込み、リランキング、リアルタイム会話などのシナリオをサポートします。
2. 汎用動画タスクプロトコルでは、タスク送信パス、照会パス、タスク ID、状態、進捗、結果 URL、エラーメッセージ、状態マッピングを設定できます。
3. リクエストボディの透過と書き換えは、チャネル設定の `Pass Through Body` と `Param Override` を再利用します。
4. 動画コンテンツは `/v1/videos/{task_id}/content` からプロキシ取得でき、アップストリームのリソースドメインを隠すことができます。

## 性能、信頼性、デプロイ

1. SQLite、MySQL、PostgreSQL をサポートします。ローカル検証には SQLite を利用でき、本番環境では MySQL または PostgreSQL を推奨します。
2. Redis / メモリキャッシュをサポートし、単一ノードと複数ノードのデプロイに対応します。
3. グローバル API / Web レート制限、重要インターフェースのレート制限、検索レート制限、モデルリクエストレート制限、グループ別モデルリクエストクォータをサポートします。
4. `STREAMING_TIMEOUT`、`STREAM_SCANNER_MAX_BUFFER_MB`、`MAX_REQUEST_BODY_MB`、`MAX_FILE_DOWNLOAD_MB` により、長いストリーミング応答、大きなリクエストボディ、リモートファイルダウンロードを制御できます。
5. アップストリーム HTTP 接続プールとタイムアウト設定、ディスクキャッシュ、Pyroscope プロファイリング、グレースフルシャットダウン、クォータキャッシュの永続化をサポートします。

## ログイン、セキュリティ、コンプライアンス

1. JWT、WebAuthn/Passkeys、OAuth、OIDC、Telegram、Discord、LinuxDO などのログイン方式をサポートします。
2. 管理者、一般ユーザー、グループ、トークン、モデルアクセス制御をサポートします。
3. リクエストボディサイズ制限、ストリーミングタイムアウト制御、エラーログ、稼働状態チェック、複数ノードで一貫したセッションシークレットをサポートします。
4. 不特定多数に生成 AI サービスを提供するデプロイ者は、届出、コンテンツ安全、実名要件、ログ保存、税務、決済、消費者保護、アップストリーム認可などのコンプライアンス事項を自ら処理する必要があります。
