MAX API の最新リリース、プレビュー、修正内容、バイナリ、イメージ情報は GitHub Releases を正としてください。

- [MAX-API-Next/MAX-API Releases](https://github.com/MAX-API-Next/MAX-API/releases)
- [MAX API Docker Hub](https://hub.docker.com/r/cscitechtop/max-api)

本番環境をアップグレードする前に、対象バージョンのリリースノートを読み、データベースと重要な設定をバックアップし、テスト環境または段階的リリースでチャネル、課金、ログ、非同期タスク、フロントエンド設定を確認してください。
