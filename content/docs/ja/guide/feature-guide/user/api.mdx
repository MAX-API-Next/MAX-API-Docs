OpenAI の `base_url` をプラットフォームアドレスに置き換え、プラットフォーム発行のトークンを `api_key` として使用するだけで呼び出しを開始できます。

## Playground オンラインテスト

左側ナビゲーションの「Playground」をクリックするか、`/console/playground` に直接アクセスしてください。

![Playground ページ](/assets/guide/feature-guide/playground.png)

Playground はコード不要でモデルと直接対話できる内蔵オンラインテストツールです。トークンが正常に動作するか確認できます。

### モデルとグループの選択

入力欄の上でモデルと利用可能なグループを選択します。`auto` または `auto:*` グループを選択すると、Playground は現在のユーザーが利用できるモデル一覧を読み込みます。実際に利用できるかどうかは、管理者が設定したチャンネル、グループ、権限に依存します。

### モデルパラメータ

入力欄のツールバーにあるパラメータボタンから設定を開きます。デスクトップではポップオーバー、モバイルでは下部シートとして表示されます。各パラメータは個別に有効化する必要があり、**有効なパラメータだけがチャット補完リクエストに送信されます**。設定は現在のブラウザに保存され、次回のテストでも利用できます。

| パラメータ | 範囲 | 用途 |
| --- | --- | --- |
| 温度（Temperature） | `0` - `2`、刻み `0.1` | ランダム性と創造性を制御します。低い値は通常より安定し、高い値はより多様になります。 |
| Top P | `0` - `1`、刻み `0.05` | 候補 Token を指定した累積確率質量に制限します。 |
| 頻度ペナルティ | `-2` - `2`、刻み `0.1` | 同じ表現の繰り返しを減らします。 |
| プレゼンスペナルティ | `-2` - `2`、刻み `0.1` | 新しい話題を取り入れるようモデルを促します。 |
| 最大トークン数 | `1` - `200000`、整数 | 応答の長さを制限します。 |
| シード | `0` - `2147483647`、整数、任意 | 対応モデルで同じリクエストの再現性を高めます。空欄の場合は `seed` を送信しません。 |

入力値は各コントロールの範囲と刻みに合わせて正規化されます。パラメータの対応状況はモデルとアップストリームサービスごとに異なるため、検証エラーが返された場合は該当パラメータを無効にするか、対応モデルを選択してください。

## API アドレスの取得

![ホームページ API アドレスコピーエリア](/assets/guide/feature-guide/api-address.png)

ホームページから API Base URL をコピーし、クライアントまたはコードの `base_url` として使用してください。

## コード例

### Python（OpenAI SDK）

```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-xxxxxxxxxxxxxxxx",  # プラットフォーム発行のトークン
    base_url="https://your-platform.com/v1"
)

response = client.chat.completions.create(
    model="gpt-4o",
    messages=[{"role": "user", "content": "Hello!"}]
)
print(response.choices[0].message.content)
```

## 対応エンドポイント

> **注**: 以下のエンドポイントは上流プラットフォーム（OpenAI、Claude、Gemini など）が提供する標準的な API エンドポイントの例です。本プラットフォームはこれらのリクエストを対応する上流サービスにプロキシします。

| エンドポイント | パス | 説明 |
| --- | --- | --- |
| チャット補完 | `POST /v1/chat/completions` | 会話生成、ストリーミング対応 |
| テキスト補完 | `POST /v1/completions` | 従来の補完エンドポイント |
| 埋め込み | `POST /v1/embeddings` | テキストのベクトル化 |
| 画像生成 | `POST /v1/images/generations` | テキストから画像 |
| 音声認識 | `POST /v1/audio/transcriptions` | Whisper など |
| 音声合成 | `POST /v1/audio/speech` | TTS |
| リランク | `POST /v1/rerank` | ドキュメントの再ランク付け |
| Responses API | `POST /v1/responses` | OpenAI Responses 形式 |
| リアルタイム | `GET /v1/realtime`（WebSocket） | OpenAI Realtime API |
| モデル一覧 | `GET /v1/models` | 利用可能なモデルの一覧 |
